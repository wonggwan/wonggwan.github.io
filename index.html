Combined Detection Approach to DNS Spoofing Attacks
Jun Wang, Yunyi Zhang, Shuyuan Jin
Spring 2019
Excellent undergraduate graduation thesis of the School of Data and Computer Science at Sun Yat-Sen University in 2019

We proposed a comprehensive DNS spoofing detection approach which is able to detect spoofing attack based on different kinds of implementation such as traditional and Kaminsky spoofing attack. We improved the calculation mechanism of passive detection into a version which calculates the difference of total packet number between response and request packets specified to each domain.

Convolutional Gated Recurrent Network for Video Matting (UPenn ESE 546 Principle of Deep Learning)
Jun Wang, Shunqi Liu
Fall 2020
Video

Conventional matting methods have poor performance when an image has similar foreground and background colors or complicated textures. Besides, current video matting algorithm does not make good use of the inter-relation (temporal logic) among frames in the video. Addressing both issues, we proposed a video matting method using Encoder-Decoder based Convolutional Neural Network (CNN) and Gated Recurrent Unit(GRU) to capture the temporal information among frames in a video.

Scale Invariant Detection (UPenn CIS 580 Machine Perception)
Jun Wang, Shunqi Liu
Spring 2020

We approximated a Laplacian of Gaussian filter (LoG) by a Difference of Gaussians (DoG) without knowing the size of the blobs to detect a priori. The LoG filter in 2D is a rotationally symmetric version of the 1D filter, which makes it a good blob detector(a blob is a dark patch on light background).

Heatmap-Based Keypoint Detection (UPenn CIS 580 Machine Perception)
Jun Wang
Spring 2020

In this project, we were required to estimate keypoints of an oilcan using a heatmap-based neural network. Each 2D heatmap corresponds to one keypoint and is targeted to localize this particular keypoint within the given image. During training, we synthesized and identified the location of each keypoint on the 2D image and used a 2D Gaussian centered on this location on the corresponding heatmap. Then we may use the keypoints detected to localize 6DoF pose of the object by aligning the CAD model of the object and the translation between object and camera frame such that the projection of the CAD model aligns with the object in the image.

Loan Defaults Detection
Jun Wang, Melanie Bigler, Jongseok Lee
Spring 2018

In this project, we analyzed 300k dataset of 9099 clients' initial information consisting of 31 elements to predict whether the new users qualified for loan using R. We conducted data classification based on logistic regression, CART decision tree, KNN algorithm and hierarchical clustering as well as non-hierarchical clustering. Concluded that data classification with KNN algorithm reached optimal effect by analyzing AUC curve of all algorithms. And we synthesized and eliminated partial ireelevant variables through PCA to enhance classification accuracy.
